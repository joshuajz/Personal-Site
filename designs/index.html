Design Notes
🎨
Shared Brand Palette

All designs use the existing colour system: #e76f51 orange, #f1dec6 peach, #0A6847 green. The redesign should feel like an evolution, not a rebrand.

📐
New Sections Added

Every design accommodates the four planned additions: Blog, Photography, Contact, and a more prominent About section — while preserving the existing Experience, Projects, and Volunteering content.

⚡
Zero-Dependency POCs

All files are standalone HTML with embedded CSS and JS. No bundler, no npm install. Open the file in a browser and it works. Production implementation would move to React/Next.js with the same visual design.
